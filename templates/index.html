How to Use
  1. Select Flags: Choose one or more feature flags to schedule targeting rules for
  2. Select Segments: Choose the segments that should be targeted by these rules
  3. Set Schedule Time: Choose when the targeting rules should be activated (in EST timezone)
  4. Choose Variation: Select which variation of the flag to serve to the segments
  5. Schedule: Click the button to create the scheduled change in LaunchDarkly
Note: The scheduled changes will be created in LaunchDarkly and will automatically execute at the specified time. You can view and manage these scheduled changes in the LaunchDarkly dashboard.
